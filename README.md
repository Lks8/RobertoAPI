Para aqueles que precisam de criatividade para exprimir seus sentimentos da maneira mais animalesca e crua possível.

Rode "node server.js" para executar o servidor e acesse localhost:3000/api/comment (a porta pode variar) para obter um comentário.

Use "node test.js" para executar os testes.

Chame RobertoAPI.getComment() para obter um comentário aleatório.

Também é possível passar o tipo "SUICIDAL" ou "ANGRY" para um comentário mais específico.
